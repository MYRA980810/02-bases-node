#Estas son los comandos que recibe la aplicacion de consola 

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Este es la base de la tabla               [number] [default: 5]
  -l, --listar   Este comando lista la tabla en consola
                                                      [boolean] [default: false]
  -h, --hasta    Este es el numero de veces por el que se multiplica el numero
                                                          [number] [default: 10]
```
